# 维基百科分析

您公司的维基百科页面是AI系统在生成有关您品牌的响应时使用的最具影响力的来源之一。 一篇得到妥善维护的文章提高了ChatGPT、Google AI Mode、Gemini、Perplexity和Copilot被准确引用的可能性。

Wikipedia Analysis机会使用AI对照行业竞争对手评估您的Wikipedia页面，并显示优先建议，以弥合对LLM可用性最重要的差距。

它跨五个维度分析您的文章：

- **引用** — 文章中引用的外部源的数目。 引用代表可信度，是LLM如何与行业平均值和最大竞争对手相比，评估Wikipedia页面权威性的关键因素。
- **节** — 文章结构和涵盖的主题范围。
- **内容长度** — 相对于行业基准的字数。
- **图像** — 文章的视觉丰富度。
- **Infobox完整性** — 结构化数据字段与竞争对手包含的内容相比存在。

![维基百科分析仪表板](/help/dashboards/opportunities/assets/wikipedia-analysis-overview.png)

## 工作原理

LLM Optimizer会清理您公司的维基百科页面，并将其与一组根据您的业务类别自动识别的行业竞争对手进行比较。 对于每个维度，它会计算与行业平均值的差距，并生成具有支持数据源的特定优先推荐。

结果显示在三个选项卡中：**建议和指导**、**市场比较**&#x200B;和&#x200B;**您的文章**。

## 建议和指导

此选项卡显示改进您的维基百科页面的战略建议。 每一项建议都包括一个优先级别，对差距的说明，它对于有限责任公司的重要性，以及修正这一差距的预期结果。

![建议和指导选项卡](/help/dashboards/opportunities/assets/wikipedia-analysis-suggestions.png)

在选项卡的顶部，**指南**&#x200B;面板提供了包含三列的高级分析摘要：

- **推荐** — 根据已识别的全套机会执行的顶级操作。
- **密钥Insight** — 总结为您的站点发现了多少改进机会。
- **基本原则** — 分析的基础，例如使用哪些行业竞争对手进行基准测试。

只有当基于实际分析数据满足相关条件时，才会显示推荐 — 例如，仅当参考计数低于行业平均值时，才会显示参考差距建议。

### 推荐类型

| 推荐 | 优先级 |
|---|---|
| 解决新闻发布音调问题 | 严重 |
| 添加参考以达到行业标准 | 严重 |
| 添加产品和服务部分 | 严重 |
| 添加信息框 | 高 |
| 增强缺少字段的信息框 | 高 |
| 添加公司历史记录部分 | 高 |
| 添加内容部分 | 高 |
| 添加领导和管理科 | 中 |
| 添加图像以符合行业标准 | 中 |
| 添加类别以提高可发现性 | 低 |
| 文章质量状态 | 信息 |

每个推荐包括：

- **描述** — 已识别间隙的简要说明。
- **为什么它很重要** — 对LLM可访问性和维基百科质量评分的影响。
- **预期结果** — 可衡量的具体结果。 例如，“添加65个以上的引用以达到行业平均水平，将引用数量增加191%”。

## 市场比较

**市场比较**&#x200B;选项卡显示竞争基准表以及对比您的维基百科页面与业界同行的可视图表。

![市场比较选项卡](/help/dashboards/opportunities/assets/wikipedia-analysis-market-comparison.png)

比较范围包括参考资料、章节和字数，帮助您了解在行业内的排名以及达到或超过基准需要做出多大改进。

## 您的文章

**您的文章**&#x200B;选项卡为您提供了当前Wikipedia页面的详细快照。

![您的文章选项卡](/help/dashboards/opportunities/assets/wikipedia-analysis-your-article.png)

它包括：

- **文章详细信息** — 行业、公司名称、网站、上次编辑日期、过去30天的编辑次数和子部分计数。
- **文章功能** — 您的文章是否具有信息框、目录、潜在客户图像，另请参阅部分和外部链接。
- **文章结构** — 当前所有部分的列表。
- **参考质量细分** — 您的参考分类（权威、行业、学术界、公司公关、其他）。
- **Infobox数据** — 当前已填充到您的Infobox中的所有字段。

## 常见问题解答

**为什么维基百科对AI 搜索很重要？**

维基百科是LLM训练数据和实时检索中最可靠的来源之一。 当人工智能系统生成关于公司的响应时，它们经常利用维基百科作为事实基础 — 创建日期、产品、领导地位、行业分类等等。 维基百科页面稀疏或结构不良，意味着您的品牌被准确引用或引用的可能性更低。

**哪些AI系统受到更强的Wikipedia页面的影响？**

改进您的维基百科页面会增加被ChatGPT（免费和付费）、Google AI概述、Google AI模式、Perplexity、Microsoft Copilot和Gemini引用的可能性。

**如何选择行业竞争对手？**

根据公司的行业分类自动识别竞争对手。 该分析最多使用六个竞争对手页面来计算基准。

**如何编辑我的维基百科页面？**

必须按照其[编辑准则](https://en.wikipedia.org/wiki/Wikipedia:Editing_policy)，直接在维基百科上编辑维基百科。 LLM Optimizer提供了您需要的特定建议和数据源 — 编辑本身在维基百科上进行。 如果您的文章被标记为存在语气问题，请在进行更改之前查看Wikipedia的[中立观点策略](https://en.wikipedia.org/wiki/Wikipedia:Neutral_point_of_view)。

**我可以直接从LLM Optimizer应用推荐吗？**

不直接 — 维基百科的编辑必须在维基百科上进行。 LLM Optimizer会确切地告诉您应该修复什么、修复什么很重要，以及在何处查找支持来源以备份您的更改。

**分析更新的频率如何？**

Wikipedia分析反映上次数据刷新时Wikipedia页面和竞争对手页面的状态。 改进后重新访问机会以跟踪您的进度。

**如果我的公司没有Wikipedia页面怎么办？**

Wikipedia分析机会需要现有的Wikipedia文章。 如果您的品牌没有这样的品牌，那么创建符合维基百科的[知名度准则](https://en.wikipedia.org/wiki/Wikipedia:Notability)的维基百科页面是值得在其他优化之前优先处理的一个基本地理步骤。
