# 情绪分析

YouTube是影响消费者认知和品牌声誉的最具影响力的平台之一。 当AI系统响应有关您品牌的提示时，它们越来越多地引用YouTube视频作为源，这使得在该内容中讨论您的品牌的方式直接输入到AI生成的响应中。

在您的品牌存在感仪表板提示集中检测到YouTube视频作为提示引文时，将显示YouTube情绪分析机会。 它分析这些被引视频及其对情绪、声音份额和重复主题的评论。 然后，它会显示优先推荐，以改进在AI生成的响应中如何感知和表示您的品牌。

它跨六个维度分析您的品牌：

- **已分析视频** — 为品牌提及和情绪检查的YouTube视频数量。
- **已分析的评论** — 已分析视频中检查的评论数。
- **品牌提及（视频）** — 在视频内容中提及您品牌的频率。
- **品牌提及（评论）** — 评论中提及您品牌的频率。
- **整体情绪（视频）** — 跨视频内容针对您的品牌的总体情绪。
- **整体情绪（评论）** — 跨评论向您的品牌聚合的情绪。

>[!NOTE]
>情绪分析当前处于测试阶段。 随着功能的不断发展，功能和可用性可能会发生变化。

![情绪分析仪表板](/help/dashboards/opportunities/assets/youtube-sentiment-overview.png)

## 工作原理

LLM Optimizer会监控AI系统引用的YouTube视频，以在品牌存在感仪表板提示集中显示提示。 当检测到引用的视频时，它会分析这些视频及其对品牌提及、情绪、声音份额和人工智能引用的评论。 它将您的品牌与市场竞争对手和相关品牌的表现进行比较，找出推动情绪的重复主题，并产生解决认知差距的建议。

如果没有为提示集中的提示引用YouTube视频，则您的仪表板中将不会显示此机会。

结果显示在两个选项卡中：**建议**&#x200B;和&#x200B;**性能**。

## 建议

此选项卡显示改善品牌在YouTube上的认知度的建议。 建议将组织为三个子选项卡： **当前建议**、**固定建议**&#x200B;和&#x200B;**忽略的建议**。

![建议选项卡](/help/dashboards/opportunities/assets/youtube-sentiment-suggestions.png)

建议表包括以下各列：

- **建议** — 建议的改进以解决感知差距。
- **优先级** — 紧急级别（严重、高、Medium、低）。
- **操作项** — 打开一个包含实施推荐的特定步骤的面板，包括负责团队（例如，内容策略、影响者营销、产品营销）。
- **证据** — 打开显示建议背后视频的源表。

展开建议会显示包含以下内容的&#x200B;**AI分析**&#x200B;部分：

- **为什么需要改进此功能** — 说明发现的认知差距，包括竞争背景以及问题在YouTube内容中的形成方式。
- **如何改进** — 有关哪些内容或操作可以解决该差距的具体指导。
- **预期结果** — 实施建议的预期结果。

**源**&#x200B;表显示了推动该建议的YouTube视频，该视频包含以下列：

- **视频** — 标题和指向YouTube视频的链接。
- **频道** — 发布视频的YouTube频道。
- **参与度** — 参与度级别（低、Medium、高）。
- **品牌提及** — 您的品牌提及次数与视频中提及的总次数。
- **声音份额** — 相对于提及的所有品牌，您的品牌提及次数所占的份额。
- **5大品牌** — 视频中提及最多的品牌。
- **情绪** — 视频中您品牌的整体情绪。
- **AI引用** — 引用此视频的AI答案数。

## 性能

“**性能**”选项卡提供了您的品牌在YouTube内容中的性能的详细细目。 它分为四个部分。

### 市场格局

根据提及内容将您的品牌与关联品牌和市场竞争对手的表现进行比较。

![市场前景](/help/dashboards/opportunities/assets/youtube-sentiment-market-landscape.png)

它显示：

- 视频中的&#x200B;**品牌提及** — 您的声音份额与相关品牌和市场竞争对手的对比。
- 评论中的&#x200B;**品牌提及** — 评论内容之间的相同比较。
- **市场跟踪** — 一个可筛选的图表，最多可以选择五个竞争对手品牌来比较视频和评论之间的声音份额。

### 情绪分析

使用&#x200B;**品牌分发**&#x200B;图表跟踪分析内容中的情绪感知，该图表显示对视频和评论有利、中性和不利的情绪的百分比细分。

![情绪分析](/help/dashboards/opportunities/assets/youtube-sentiment-distribution.png)

### 视频

分析过的YouTube视频的详细表格，其中包含以下列：

- **视频** — 标题和指向YouTube视频的链接。
- **频道** — 发布视频的YouTube频道。
- **参与度** — 参与度级别（低、Medium、高）。
- **品牌提及** — 您的品牌提及次数与视频中提及的总次数。
- **声音份额** — 相对于提及的所有品牌，您的品牌提及次数所占的份额。
- **5大品牌** — 视频中提及最多的品牌。
- **情绪** — 视频中您品牌的整体情绪。
- **AI引文** — 与视频关联的AI引文信号数。

“性能”选项卡在一个视图中显示&#x200B;**视频**&#x200B;和&#x200B;**主题**&#x200B;面板（已选择&#x200B;**视频**）。 下图包含视频级别的表，其下方为&#x200B;**主题**&#x200B;摘要。

## 常见问题解答

**为什么YouTube对AI 搜索很重要？**

在生成有关品牌、产品和主题的响应时，AI系统越来越多地引用YouTube视频。 当这些引用视频以不利或不准确的方式讨论您的品牌时，该情绪将直接涉及AI系统如何代表您的品牌。 改善AI系统已引述的YouTube内容中讨论品牌的方式是影响AI生成的品牌认知的最直接方式之一。

**为什么我的信息板中未显示此机会？**

只有在检测到YouTube视频为品牌存在感仪表板提示集中的提示引文时，才会显示此机会。 如果没有为这些提示引用YouTube视频，则不会显示机会。 随着您的品牌获得更多的YouTube报道，并且这些视频被AI系统引用于您的提示集，机会将变得可用。

**整体情绪是什么意思？**

整体情绪反映了提及您的品牌时内容的整体语调：有利、中立或不利。 由于视频和评论可能存在显着差异，因此会单独计算视频和评论。

**什么是声音份额？**

声音份额是指您的品牌在给定内容或所有已分析内容中的品牌提及总数中所占的百分比（相对于提及的所有其他品牌）。

**什么是AI引文？**

人工智能引文显示给定视频有多少人工智能答案。 人工智能引文计数较高表明，在生成有关相关主题的响应时，人工智能系统正在积极使用视频，这使得这些视频中的情绪对于您品牌的AI呈现尤为重要。

**如何识别市场竞争对手？**

可根据您品牌的行业以及分析内容中最常提到的品牌自动识别竞争对手。 您最多还可以手动选择五个品牌以在市场跟踪图中进行比较。

**分析更新的频率如何？**

YouTube分析反映分析到的内容是仪表板标题中显示的日期。 执行建议后重新访问机会以跟踪情绪和声音份额中的更改。
