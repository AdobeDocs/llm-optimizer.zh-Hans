# 引用情绪分析

当AI系统回答有关您品牌的问题时，它们依赖一组最常引用的URL — 在AI生成的响应中经常引用的第三方网页。 如何在这些页面上描绘您的品牌将直接影响AI系统向用户呈现品牌的方式。

引证情绪分析机会分析在品牌存在感仪表板提示集中检测到的提示最常引用的URL。 它评估这些页面中的品牌提及、情绪、声音份额和重复主题。 之后，它会显示优先推荐，以改进人工智能系统最依赖的内容对您品牌的认知方式。

它包含四个关键量度：

- **已分析的网页数** — 已检查用于品牌提及和情绪的被引用网页数。
- **跳过的页数** — 无法分析的页数（例如，由于访问限制）。
- **品牌提及（页面）** — 在分析页面中提及您品牌的频率。
- **整体情绪（页面）** — 跨分析页面汇总您的品牌情绪。

>[!NOTE]
>所引用的情绪分析当前处于测试阶段。 随着功能的不断发展，功能和可用性可能会发生变化。

![引用的情绪分析仪表板](/help/dashboards/opportunities/assets/cited-sentiment-overview.png)

## 工作原理

LLM Optimizer标识在AI生成的响应中为品牌存在感仪表板提示集中的提示出现的最常引用的URL。 它针对品牌提及、情绪、声音份额和人工智能引文分析这些页面。 它可将您的品牌与市场竞争对手进行比较，识别重复出现的主题，并生成建议以解决对AI系统最重要的页面上的认知缺口。

如果没有在您的提示集中检测到提示的引用URL，则您的仪表板中将不会显示此机会。

结果显示在两个选项卡中：**建议**&#x200B;和&#x200B;**性能**。

## 建议

此选项卡显示用于改善品牌在主要引用URL中的认知度的建议。 建议将组织为三个子选项卡： **当前建议**、**固定建议**&#x200B;和&#x200B;**忽略的建议**。

![建议选项卡](/help/dashboards/opportunities/assets/cited-sentiment-suggestions.png)

建议表包括以下各列：

- **建议** — 建议的改进以解决感知差距。
- **优先级** — 紧急级别（严重、高、Medium、低）。
- **操作项** — 打开一个包含实施推荐的特定步骤的面板，包括负责团队。

展开建议会显示包含以下内容的&#x200B;**AI分析**&#x200B;部分：

- **为什么需要改进** — 说明识别出的认知差距，包括哪些引用的URL不能充分代表您的品牌和竞争环境。
- **如何改进** — 关于外联、内容创建或合作行动的具体指导意见以解决差距。
- **预期结果** — 实施建议的预期结果。

## 性能

**性能**&#x200B;选项卡提供您的品牌在排名最前的页面上的表现的详细细目。 它分为四个部分。

### 市场格局

根据所引用页面中的提及内容，将您的品牌与关联品牌和市场竞争对手的表现进行比较。

![市场前景](/help/dashboards/opportunities/assets/cited-sentiment-landscape.png)

它显示：

- **个页面中的品牌提及** — 您的声音份额与关联品牌和市场竞争对手的对比。
- **市场跟踪** — 一个可筛选图表，最多可以选择五个竞争对手品牌来比较分析页面中的声音份额。

### 情绪分析

使用&#x200B;**品牌分布**&#x200B;图表跟踪分析页面上的情绪感知，该图表显示各页面中有利、中立和不利情绪的百分比细分。

![情绪分析](/help/dashboards/opportunities/assets/cited-sentiment-distribution.png)

### 页面

已分析引用网页的详细表格，包含以下列：

- **页面** — 分析页面的URL。
- **品牌提及** — 您的品牌提及次数与页面上提及的总次数。
- **声音份额** — 相对于提及的所有品牌，您的品牌提及次数所占的份额。
- **5大品牌** — 页面上提及最多的品牌。
- **情绪** — 页面上的品牌整体情绪。
- **AI引用** — 引用此页面的AI答案数。

### 主题

在分析页面中标识的重复主题的表，其中显示：

- **主题** — 已识别周期性主题或主题。
- **品牌提及** — 与主题相关的品牌提及数。
- **情绪** — 与主题关联的整体情绪。

单击任何主题上的&#x200B;**详细信息**&#x200B;将打开一个包含分析摘要和贡献源页面的深入分析。

## 常见问题解答

**为什么最常被引用的URL对AI 搜索很重要？**

最常引用的URL是AI系统在生成有关您品牌的响应时最常引用的第三方页面。 这些页面上的情绪和框架对AI如何代表您的品牌有着直接和巨大的影响，其影响甚至超过很少被引用的页面。 改进在这些特定页面上展示您的品牌的方式，是您可以采取的最有效操作之一，以便提高AI可见性。

**为什么我的信息板中未显示此机会？**

仅当在您的品牌存在感仪表板提示集中检测到提示的引用URL时，才会出现此机会。 如果没有为这些提示标识引用的URL，则不会显示机会。

**跳过的页面是什么意思？**

跳过的页面被引用为无法分析的URL，通常是因为页面位于收费墙之后、需要身份验证或阻止自动访问。 这些页面将被计数，但不会计入情绪和品牌提及分析。

**什么是声音份额？**

声音份额是指您的品牌在给定页面或所有已分析页面上的品牌提及总数中所占的百分比（相对于提及的所有其他品牌）。

**什么是AI引文？**

人工智能引文显示给定页面引用了多少人工智能答案。 人工智能引文计数较高表明，在生成有关相关主题的响应时，人工智能系统正在积极使用该页面，这使得这些页面上的情绪对于您品牌的AI呈现尤为重要。

**如何识别市场竞争对手？**

可根据您品牌的行业以及在分析页面中最常提及的品牌自动识别竞争对手。 您最多还可以手动选择五个品牌以在市场跟踪图中进行比较。

**分析更新的频率如何？**

该分析反映截至仪表板标题中显示之日检测到的引用URL。 执行建议后重新访问机会以跟踪情绪和声音份额中的更改。
