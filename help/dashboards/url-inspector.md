# URL 检查器

URL 检查器可帮助您分析域名下特定页面在 AI 搜索中的表现。它在 URL 层级整合可见度、代理式流量和引荐数据，为您提供精细化视图，展示哪些 URL 得到引用以及它们在回答中出现的频率。

![URL 检查器](/help/dashboards/assets/url-insp.png)

## 筛选条件

在页面顶部，您可以应用筛选条件以优化视图。您选择的筛选条件将影响仪表板中的&#x200B;**所有**&#x200B;部分。您可以自定义以下内容：

* **日期范围** ：选择显示数据的时间范围。例如，最近 4 周。您也可以选择&#x200B;**自定义周数**&#x200B;选项来自定义时间周期。
* **类别** ：按类别筛选显示结果。
* **平台**：选择要分析的 AI 引擎。
* **页面内容类型** ：按内容类型进行筛选。
* **区域** ：按地理位置筛选结果。并非所有区域在发布初期均可用。

选择所需筛选条件后，单击&#x200B;**应用筛选条件**&#x200B;以将其应用到仪表板。

## 概述量度

URL 检查器提供多项概述量度，帮助您快速评估页面在 AI 搜索中的表现。提供以下量度：

* **含自有品牌引用的独特提示词**：包含自有品牌引用的独特 AI 提示词总数。
* **独特提示词的总数**：独特 AI 提示词的总数量。
* **已引用的独特 URL**：已引用的独特自有 URL 数量。
* **引用总次数**：自有 URL 在 AI 生成的回答中已引用的总次数。
* **代理总点击数**：AI 代理点击您的 URL 的总次数。
* **来自 LLM 的引荐点击数**：从 AI 生成回答跳转至您 URL 的总点击次数。

每个概述量度均配有趋势指示器，用于显示相较于上一时间段的变化情况。

## 您被引用的 URL

“已引用 URL”视图列出所有在 AI 生成的回答中引用的品牌 URL，并附带相关量度。两个表格均提供搜索字段以便快速查找主题，您还可以通过单击&#x200B;**配置列**&#x200B;按钮来自定义显示的量度。此外，您可以使用&#x200B;**导出**&#x200B;选项下载表格的 .csv 文件，与团队共享洞察，或将其纳入管理层报告。

![引用的 URL](/help/dashboards/assets/cited-urls.png)

提供以下量度：

* **URL**：分析的 URL。
* **引用的次数**：AI 生成的回答中引用该 URL 的次数。
* **引用的提示词数量**：引用了该 URL 的独特 AI 提示词数量。
* **类别**：与该 URL 关联的产品类别或主题。
* **区域**：引用该 URL 的地理区域。
* **代理点击数**：AI 代理对该 URL 的总点击数。
* **引荐点击数**：从 AI 生成的回答跳转至该 URL 的点击数。

## 正在争夺引用的热门 URL

“正在争夺引用的热门 URL”视图会突出显示当前在与您品牌相关的回答中出现引用的外部 URL，用于衡量在您所在领域中谁获得更多引用。该数据表提供搜索字段，方便快速定位特定 URL。此外，您可以使用&#x200B;**导出**&#x200B;选项下载表格的 .csv 文件，与团队共享洞察，或将其纳入管理层报告。

![正在争夺引用的热门 URL](/help/dashboards/assets/trend-url.png)

提供以下量度：

* **URL**：分析的 URL。
* **内容类型**：内容类型（自有、社交、获得、其他）。
* **引用的次数**：AI 生成的回答中引用该 URL 的次数。
* **引用的提示词数量**：引用了该 URL 的独特 AI 提示词数量。
* **类别**：与该 URL 关联的产品类别或主题。
* **区域**：引用该 URL 的地理区域。

### 详细信息窗口

在“已引用”和“热门”两个视图中，每行 URL 的末尾都有一个&#x200B;**详细信息**&#x200B;按钮。点击该按钮会打开一个单独窗口，显示更多详细信息。详细信息窗口会显示该 URL 的引用频率、<!--the sentiment of AI responses where it is mentioned,-->出现在其中的主题与提示词，以及随时间变化的代理式流量与引荐流量趋势（适用于自有 URL）。

![详细信息窗口](/help/dashboards/assets/details-url.png)
